## 1. 设计基线

- **目标平台**：iOS App。
- **设计规范**：全部 UI 以 **iOS 26 设计规范（HIG）** 为准，包括导航、组件、间距、字体、动效与配色。

## 2. 全局导航结构

参照 iOS 26 原生底部导航形态（药丸 Tab 组 + 右侧独立圆 + 常驻 BottomAccessory 条，整体对齐最新版 Apple 播客的三状态，详见 DESIGN §5.2）：

- **顶栏**：左侧为页面标题，右上角为用户头像 → 进入「我的 / 设置」。
- **底部导航**（三个角色）：
  - **药丸**：首页 / 报表 / 家庭（3 个导航 Tab）。
  - **右侧独立 Search 圆**（🔍，`role="search"`）：点击进入搜索（流水 / 分类 / 成员）。
  - **记一笔**：常驻 **BottomAccessory 三栏条**（今日配图 / 今日格言 / ➕），整条可点 → 弹出记账面板（大号 detent Sheet）；**不再是右侧独立 ➕ 圆**。
- **三状态**：常态两行堆叠（记一笔条在上、药丸+Search 圆在下）／搜索激活（药丸变搜索框、记一笔条隐藏）／滚动合并一行。

```
┌─────────────────────────────┐
│  首页                [头像] │  ← 顶栏（标题 + 右上角头像）
│                             │
│         （页面内容）         │
│                             │
├─────────────────────────────┤
│  [ 🖼 配图    今日格言 ↔        ➕ ] │  ← 记一笔 BottomAccessory 三栏条（满宽）
├─────────────────────────────┤
│  ( 🏠首页  📊报表  👨‍👩‍👧家庭 )   ( 🔍 ) │  ← 药丸 3 Tab（左）+ Search 独立圆（右）
└─────────────────────────────┘
```

| 区域     | 元素                                     | 作用                       |
| -------- | ---------------------------------------- | -------------------------- |
| 顶栏右上 | 头像                                     | 进入「我的 / 设置」        |
| 底栏药丸 | 首页 / 报表 / 家庭（3 Tab）              | 主导航                     |
| 底栏右侧 | 🔍 Search 独立圆                         | 搜索（流水 / 分类 / 成员） |
| 记一笔条 | BottomAccessory 三栏（配图 / 格言 / ➕） | 记一笔（主操作），整条可点 |

## 5. 说明与待定

- 预算、储蓄目标到 P1 时，可作为「家庭」页内的入口或新增 Tab；**届时底部是否扩成 4 Tab 再定**（注意 iOS 26 下 Tab 数量、右侧 Search 圆与记一笔条的布局平衡）。
- 「扫一扫」入口建议同时放在「家庭页」和「我的页」。
- 顶部条幅（预算预警 / 月度总结提醒）属 P1，P0 阶段首页顶部保持简洁。
